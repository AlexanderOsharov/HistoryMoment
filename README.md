# Проект "Интерактивная карта культурных объектов Москвы"

## О проекте
Проект "Интерактивная карта культурных объектов Москвы" посвящен демонстрации исторических мест Москвы и развитию интереса к местному культурному наследию. Подробнее с проектом можно ознакомиться в документации [4].
